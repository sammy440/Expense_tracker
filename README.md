# Personal Expense Tracker

A desktop-based Personal Expense Tracker built with Python and CustomTkinter.

## Features
- **Add Expense**: Categorize and date your expenses.
- **View Expenses**: Scrollable table with sorting and filtering.
- **Dashboard**: Visual analytics with charts.
- **Local Storage**: Data saved in `expenses.json`.

## Setup
1. Install dependencies:
   ```bash
   pip install customtkinter CTkTable matplotlib pillow tkcalendar
   ```
2. Run the application:
   ```bash
   python main.py
   ```
